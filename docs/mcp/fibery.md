# Fibery MCP

## Description
Perform queries and entity operations in your Fibery workspace.

## Repository
Celebrum/servers

## Configuration
Add to `Subrepos` list in `modulesettings.json`:
```json
"Subrepos": [
  "servers",
  "containerz-plugin",
  "models-prefab",
  "tensorzero",
  "fibery"
]
```
