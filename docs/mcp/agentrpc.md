# AgentRPC MCP

* **Description**: Connect to any function, any language, across network boundaries using AgentRPC.
* **Repository**: Celebrum/servers
* **Configuration**:
  - Add to `Subrepos` list in `modulesettings.json`:
    ```json
    "Subrepos": [
      "servers",
      "containerz-plugin",
      "models-prefab",
      "tensorzero",
      "agentrpc"
    ]
    ```
