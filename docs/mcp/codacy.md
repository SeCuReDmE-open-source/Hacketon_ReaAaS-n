# Codacy MCP

## Description
Interact with Codacy API to query code quality issues, vulnerabilities, and coverage insights about your code.

## Repository
Celebrum/servers

## Configuration
Add to `Subrepos` list in `modulesettings.json`:
```json
"Subrepos": [
  "servers",
  "containerz-plugin",
  "models-prefab",
  "tensorzero",
  "codacy"
]
```
