# CodeLogic MCP

## Description
Interact with CodeLogic, a Software Intelligence platform that graphs complex code and data architecture dependencies, to boost AI accuracy and insight.

## Repository
Celebrum/servers

## Configuration
Add to `Subrepos` list in `modulesettings.json`:
```json
"Subrepos": [
  "servers",
  "containerz-plugin",
  "models-prefab",
  "tensorzero",
  "codelogic"
]
```
