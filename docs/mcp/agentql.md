# AgentQL MCP

## Description
Enable AI agents to get structured data from unstructured web with AgentQL.

## Repository
Celebrum/servers

## Configuration
Add to `Subrepos` list in `modulesettings.json`:
```json
"Subrepos": [
  "servers",
  "containerz-plugin",
  "models-prefab",
  "tensorzero",
  "agentql"
]
```
