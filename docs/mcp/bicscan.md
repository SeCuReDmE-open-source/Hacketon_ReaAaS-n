# BICScan MCP

## Description
Risk score / asset holdings of EVM blockchain address (EOA, CA, ENS) and even domain names.

## Repository
Celebrum/servers

## Configuration
Add to `Subrepos` list in `modulesettings.json`:
```json
"Subrepos": [
  "servers",
  "containerz-plugin",
  "models-prefab",
  "tensorzero",
  "bicscan"
]
```
