# Financial Datasets MCP

## Description

Stock market API made for AI agents.

## Repository

Celebrum/servers

## Configuration

Add to `Subrepos` list in `modulesettings.json`:

```json
"Subrepos": [
  "servers",
  "containerz-plugin",
  "models-prefab",
  "tensorzero",
  "financial-datasets"
]
```
