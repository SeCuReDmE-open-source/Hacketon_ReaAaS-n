### Elasticsearch

* **Description**: Query your data in Elasticsearch.
* **Repository**: Celebrum/servers
* **Configuration**:
  - Add to `Subrepos` list in `modulesettings.json`:
    ```json
    "Subrepos": [
      "servers",
      "containerz-plugin",
      "models-prefab",
      "tensorzero",
      "elasticsearch"
    ]
    ```
