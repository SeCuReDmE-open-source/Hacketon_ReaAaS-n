# Bankless Onchain MCP

## Description
Query Onchain data, like ERC20 tokens, transaction history, smart contract state.

## Repository
Celebrum/servers

## Configuration
Add to `Subrepos` list in `modulesettings.json`:
```json
"Subrepos": [
  "servers",
  "containerz-plugin",
  "models-prefab",
  "tensorzero",
  "bankless-onchain"
]
```
