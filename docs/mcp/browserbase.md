### Browserbase

* **Description**: Automate browser interactions in the cloud (e.g. web navigation, data extraction, form filling, and more).
* **Repository**: Celebrum/servers
* **Configuration**:
  - Add to `Subrepos` list in `modulesettings.json`:
    ```json
    "Subrepos": [
      "servers",
      "containerz-plugin",
      "models-prefab",
      "tensorzero",
      "browserbase"
    ]
    ```
