### Apache IoTDB

* **Description**: MCP Server for Apache IoTDB database and its tools.
* **Repository**: Celebrum/servers
* **Configuration**:
  - Add to `Subrepos` list in `modulesettings.json`:
    ```json
    "Subrepos": [
      "servers",
      "containerz-plugin",
      "models-prefab",
      "tensorzero",
      "apache-iotdb"
    ]
    ```
