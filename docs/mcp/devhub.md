# DevHub MCP

## Description
Manage and utilize website content within the DevHub CMS platform.

## Repository
Celebrum/servers

## Configuration
Add to `Subrepos` list in `modulesettings.json`:
```json
"Subrepos": [
  "servers",
  "containerz-plugin",
  "models-prefab",
  "tensorzero",
  "devhub"
]
```
