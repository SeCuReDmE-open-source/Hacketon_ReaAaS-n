# Fewsats MCP

## Description
Enable AI Agents to purchase anything in a secure way using Fewsats.

## Repository
Celebrum/servers

## Configuration
- Add to `Subrepos` list in `modulesettings.json`:
  ```json
  "Subrepos": [
    "servers",
    "containerz-plugin",
    "models-prefab",
    "tensorzero",
    "fewsats"
  ]
  ```
