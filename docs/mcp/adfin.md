### Adfin

* **Description**: The only platform you need to get paid - all payments in one place, invoicing and accounting reconciliations with Adfin.
* **Repository**: Celebrum/servers
* **Configuration**:
  - Add to `Subrepos` list in `modulesettings.json`:
    ```json
    "Subrepos": [
      "servers",
      "containerz-plugin",
      "models-prefab",
      "tensorzero",
      "adfin"
    ]
    ```
