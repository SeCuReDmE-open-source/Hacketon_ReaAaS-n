# Axiom MCP

## Description
Query and analyze your Axiom logs, traces, and all other event data in natural language.

## Repository
Celebrum/servers

## Configuration
Add to `Subrepos` list in `modulesettings.json`:
```json
"Subrepos": [
  "servers",
  "containerz-plugin",
  "models-prefab",
  "tensorzero",
  "axiom"
]
```
