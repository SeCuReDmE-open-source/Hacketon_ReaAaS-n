### EduBase

* **Description**: Interact with EduBase, a comprehensive e-learning platform with advanced quizzing, exam management, and content organization capabilities.
* **Repository**: Celebrum/servers
* **Configuration**:
  - Add to `Subrepos` list in `modulesettings.json`:
    ```json
    "Subrepos": [
      "servers",
      "containerz-plugin",
      "models-prefab",
      "tensorzero",
      "edubase"
    ]
    ```
