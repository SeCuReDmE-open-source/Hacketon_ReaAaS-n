# Chargebee MCP

## Description
MCP Server that connects AI agents to Chargebee platform.

## Repository
Celebrum/servers

## Configuration
Add to `Subrepos` list in `modulesettings.json`:
```json
"Subrepos": [
  "servers",
  "containerz-plugin",
  "models-prefab",
  "tensorzero",
  "chargebee"
]
```
