# eSignatures MCP

## Description
Contract and template management for drafting, reviewing, and sending binding contracts.

## Repository
Celebrum/servers

## Configuration
Add to `Subrepos` list in `modulesettings.json`:
```json
"Subrepos": [
  "servers",
  "containerz-plugin",
  "models-prefab",
  "tensorzero",
  "esignatures"
]
```
