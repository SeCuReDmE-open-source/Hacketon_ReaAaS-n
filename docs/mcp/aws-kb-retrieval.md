* **Repository**: Celebrum/servers
* **Configuration**:
  - Add to `Subrepos` list in `modulesettings.json`:
    ```json

      "containerz-plugin",
      "models-prefab",
      "tensorzero",
      "aws-kb-retrieval"
    ]
    ```
