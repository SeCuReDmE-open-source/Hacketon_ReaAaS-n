# Chroma MCP

* **Description**: Embeddings, vector search, document storage, and full-text search with the open-source AI application database.
* **Repository**: Celebrum/servers
* **Configuration**:
  - Add to `Subrepos` list in `modulesettings.json`:
    ```json
    "Subrepos": [
      "servers",
      "containerz-plugin",
      "models-prefab",
      "tensorzero",
      "chroma"
    ]
    ```
